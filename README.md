# Analyse-de-sentiment-
Analyse de sentiment sur des avis de prise de médicaments / NLP 

# Description

Analyse utilisant différentes méthodes de machine learning. On a aussi recours au Natural Language Processing (NLP) pour l'nalayse de nos données textuelles.



## Dataset

Les données ne sont pas incluses dans ce dépôt.  
Téléchargez-les depuis Kaggle :
[Drug Review Dataset - Kaggle](https://www.kaggle.com/datasets/jessicali9530/kuc-hackathon-winter-2018)

Placez le fichier dans le dossier `/data`.

- 161 297 données d'entraînement
- 53 766 données de test

Variables des data set : 'uniqueID', 'drugName', 'condition', 'review', 'rating', 'date', 'usefulcount'
